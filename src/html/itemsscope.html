Mobile Items Scope

A React Native implementation of OneDrive. To be implemented within other apps to view OneDrive files.

Mobile Items Scope Goals

At Microsoft, my role as an Explore Intern was to create the PM Spec presentation, and then develop the project along with my two teammates. My goals for this project were:

  • Create the bottom sheet to show the actions available for the item.
  • Create the action that Creates a Folder.
  • Create the action that Renames a File or Folder.

The purpose of the Mobile Items Scope is to place a "Lite" version of OneDrive into other applications. This allows users to browse their OneDrive or SharePoint files without leaving the app, or needing to install the OneDrive app.
